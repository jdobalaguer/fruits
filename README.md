
### Keyboard

**`[§]`** or **`[ECS`]** : Cancel and exit  
**`[Rteurn`]** : Move forward during breaks  
**`[T]`** : Trigger  
**`[H]`** or **`[A]`** : Target response  
**`[J]`** or **`[B]`** : Non-target response  

### Tasks

**`RS`** : See two images, say if they correspond to the same day (target, match) or not.  
**`VI`** : See three images, say if they are part of the training set (target, valid) or not.  

### Timings

**`RS`** : 3 scanner runs of 817 seconds  
**`VI`** : 3 scanner runs of 385 seconds
